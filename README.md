# Calculator Application\n\n## Overview\nThis is a simple web-based calculator application that allows users to perform basic arithmetic operations.\n\n## Setup Instructions\n1. Clone the repository: `git clone <repository-url>`\n2. Navigate to the `backend` directory: `cd backend`\n3. Install the required packages: `pip install -r requirements.txt`\n4. Run the FastAPI server: `uvicorn main:app --reload`\n\n## Frontend Instructions\n1. Open `frontend/index.html` in your web browser.\n2. Input two numbers, select an operation, and click 'Calculate' to see the result.\n3. Use the 'Reset' button to clear the inputs.
